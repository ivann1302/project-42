Создай статический HTML сайт с фокусом на максимальную производительность и SEO.


ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ:


1. ПРОИЗВОДИТЕЛЬНОСТЬ (100/100 PageSpeed):
 • **LCP цель**: <2.5s (изображение hero или H1) - **FID/INP цель**: <100ms (минимизация JS на первом экране) - **CLS цель**: <0.1 (фиксированные размеры для всех элементов, включая шрифты)
   • Встроить ВСЕ критические CSS в <style> в <head> (только стили для первого экрана)
  • Считай первым экраном: header + hero + CTA (до 800px по высотен десктопе и 70vdh на смартфонах)
  • В критический CSS включать ТОЛЬКО стили этих блоков[a][b]
   • Неиспользуемые CSS ниже первого экрана загружать асинхронно: <link rel="stylesheet" href="styles.css" media="print" onload="this.media='all'">
   • ВСЕ изображения: AVIF с WebP/JPEG фолбэк через <picture>, lazy loading, decoding="async", к каждой картинке задай числовые размеры width/height в пикселях, 
• Для всех <img> использовать srcset и sizes
• sizes рассчитывать исходя из container max-width
• Responsive breakpoints: 320, 640, 768, 1024, 1280, 1920 
• Blur placeholder или LQIP (Low Quality Image Placeholder) 
• aspect-ratio в CSS для предотвращения layout shift
• speakable разметка для голосового поиска
• Для статики: Cache-Control: public, max-age=31536000, immutable
• Для HTML: max-age=0, must-revalidate
• В инструкции для сервера указать Brotli (br) приоритетно, gzip fallback
  • Предзагрузить LCP-изображение: <link rel="preload" as="image" href="hero.webp" fetchpriority="high">
   • Для главного изображения добавить fetchpriority="high"
   • ВСЕ скрипты (если есть) должны иметь атрибут defer и располагаться перед </body>
   • Абсолютные пути для ВСЕХ ресурсов: src="https://site.ru/images/photo.webp"


2. HTML СТРУКТУРА:
   • Чистый семантический HTML5: <header>, <nav>, <main>, <article>, <section>, <footer>
   • Один адаптивный HTML без дублей контента (mobile/desktop)
   • Иерархия заголовков: один H1, далее H2-H6 по логике
   • Язык сайта в зависимости от текста указать в формате: <html lang="ru-RU"> или аналогичном для другого языка
   • Viewport: <meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=5.0, viewport-fit=cover">
 • Сжатие HTML, CSS и JS файлов: минификация кода, удаление комментариев и лишних пробелов




3. SEO ОПТИМИЗАЦИЯ:
   • Все мета-теги:
     - <title>Уникальный заголовок до 60 символов </title>
     - <meta name="description" content="Уникальное описание до 160 символов с ключевыми словами">
     - <meta name="robots" content="index, follow, max-image-preview:large, max-snippet:-1, max-video-preview:-1">
     - <link rel="canonical" href="https://site.ru/">
   
   • Open Graph разметка:
     - og:title, og:description, og:image, og:type, og:url, og:locale

• Twitter card разметка:
     - twitter:card, twitter:title, twitter:description, twitter:image, twitter:url 


   • JSON-LD структурированные данные (в конце body):
     - @type: WebSite + Organization/LocalBusiness (с GEO данными: адрес, телефон, координаты)
     - @type: BreadcrumbList
    - если есть в контенте такие блок то добавь мироразметку для FAQPage и отзывов
     - Все URL абсолютные, @id указан


4. БЕЗОПАСНОСТЬ И ДОСТУПНОСТЬ:
   • <meta name="referrer" content="strict-origin-when-cross-origin">
   • Заголовки безопасности (в инструкции для .htaccess):
     X-Content-Type-Options: nosniff
     X-Frame-Options: DENY
     Permissions-Policy: camera=(), microphone=(), geolocation=()
   
   • Доступность: aria-атрибуты для интерактивных элементов,
 • Title и alt обязателен всех изображений соответствующий названию блока и номеру в блоке если изображений несколько
   • Все внешние ссылки оберни в инструкцию:<a href="https://example.com" target="_blank" rel="noopener noreferrer">Внешний сайт</a>
   • Все ссылки на файлы и внешние ссылки указывай с безопасным протоколом (HTTPS): Всегда используйте ссылки, начинающиеся с https://

5. CSS/ШРИФТЫ:
   • ТОЛЬКО системные шрифты: font-family: system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", "Roboto", "Oxygen", "Ubuntu", "Cantarell", "Helvetica Neue", Arial, sans-serif;
   • Запрещены: внешние шрифты, font-display: swap, Google Fonts
   • Предотвращение CLS:
     * { box-sizing: border-box; }
     img { max-width: 100%; height: auto; display: block; }
     .container { width: 100%; max-width: 1200px; margin: 0 auto; padding: 0 15px; }


6. ЗАПРЕЩЕНО ИСПОЛЬЗОВАТЬ:
   • Внешние JS библиотеки (jQuery, React, Vue, etc.)
   • Внешние CSS фреймворки (Bootstrap, Tailwind)
   • SVG изображения
   • Внешние шрифты
   • iframe (кроме карт, но тогда с loading="lazy")
   • document.write(), синхронные скрипты


7. ТЕСТИРОВАНИЕ:
   • Валидный HTML по W3C
   • Корректное отображение на всех размерах от 320px до 1920px
   • Поддержка всех современных браузеров


8. ДОСТУПНОСТЬ И ИНКЛЮЗИВНОСТЬ:
   • Соответствие WCAG 2.1 Level AA
   • Контрастность текста минимум 4.5:1
   • Поддержка prefers-reduced-motion
   • Все интерактивные элементы должны быть доступны с клавиатуры
