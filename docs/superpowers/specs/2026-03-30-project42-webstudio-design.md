# Project 42 — Сайт веб-студии: Design Spec

**Дата:** 2026-03-30
**Проект:** project-42 (Next.js 15, FSD, SCSS Modules)
**Язык:** Русский

---

## 1. Позиционирование и нарратив

### Главный посыл

**"Не ещё одна веб-студия."**
Project 42 — партнёр, который решает задачи бизнеса и остаётся рядом, когда задачи меняются.

### Нарратив страницы (Вариант A — Проблема → Решение)

Hero с провокацией → блок отличий от конвейерных студий → услуги → процесс → портфолио → цифры → отзывы → цены → CTA.

### Ключевые УТП

1. **Мало проектов** — не более 5 одновременно, каждый получает полное внимание
2. **AI как инструмент** — нейросети ускоряют рутину, не заменяют экспертизу; результат: ниже цена, не ниже качество
3. **Отлаженные процессы** — выверенные процессы и AI-инструменты дают скорость и качество большой студии при стоимости небольшой команды
4. **Полный цикл** — дизайн, разработка, SEO, GEO, поддержка — всё под одной крышей, без потерь на стыках
5. **Код без замков** — чистый, документированный код; другой специалист подхватит проект без боли

### Целевая аудитория

Малый и средний бизнес, оба сегмента. Приоритет — клиенты, ориентированные на долгосрочное сотрудничество, а не разовый заказ.

## 2. Визуальная система

Стиль строго следует референсам (`desktop.webp`, `mobile.webp`): тёмная тема, фиолетово-синий акцент.

### Цвета (уже в `variables.scss`)

```scss
$color-bg: #080812; // основной фон
$color-bg-alt: #0a0a1a; // чередующийся фон секций
$color-card: #12122a; // фон карточек
$color-accent: #7c5cf6; // фиолетовый акцент (primary)
$color-accent-hover: #4f8ef7; // синий на hover
$color-text: #ffffff;
$color-text-muted: #6060a0;
$color-border: rgba(255, 255, 255, 0.06);
```

### Типографика

- **Display / H1** — крупный, uppercase, display-шрифт (2 строки в Hero)
- **Eyebrow** — маленький uppercase, letter-spacing 0.15em, цвет акцента
- **Body** — body-шрифт, 16px базовый
- **Карточки** — заголовок ~20px, описание ~14px muted

### Компоненты (переиспользуемые из shared/ui)

- `Button` — primary (фиолетовый fill) / ghost (прозрачный, белая обводка)
- `SectionTitle` — eyebrow + h2, поддержка `light` варианта
- `Container` — max-width 1440px
- `Icon` — inline SVG реестр
- `Modal` — для формы заявки

## 3. Секции страницы

### 3.1 Header

- Логотип: `Project 42` (текст или SVG-логотип)
- Навигация: Услуги / Как работаем / Портфолио / Цены
- CTA: кнопка `Обсудить проект` (primary, sm)
- Поведение: прозрачный на старте, тёмный `#0A0A1A` с backdrop-blur при скролле
- Мобильный: бургер-меню

### 3.2 Hero

- Eyebrow: `— Project 42`
- H1: `Не ещё одна веб-студия` (две строки, крупный display)
- Подзаголовок: `Решаем задачи бизнеса. Остаёмся рядом, когда задачи меняются.`
- CTA-группа: `Обсудить проект` (primary, lg) + `Посмотреть работы` (ghost, lg)
- Декор: градиентное свечение фиолетово-синее в центре/снизу (как в референсе)
- Фон: `$color-bg` (#080812)

### 3.3 Почему мы другие

- Eyebrow: `Наш подход`
- H2: `Мы устроены иначе`
- Сетка: 5 карточек (на десктопе — 3+2 или 2+3, на мобильном — 1 колонка)
- Каждая карточка: иконка + заголовок + 2–3 строки описания
- Фон секции: `$color-bg-alt`

Содержимое карточек:
| # | Заголовок | Описание |
|---|---|---|
| 1 | Мало проектов | Одновременно в работе не более 5 проектов. Каждый получает максимальное внимание команды. |
| 2 | AI как инструмент | Нейросети ускоряют рутину, не заменяют экспертизу. Ниже цена — не значит ниже качество. |
| 3 | Отлаженные процессы | Выверенные процессы и AI-инструменты дают скорость и качество большой студии при стоимости небольшой команды. |
| 4 | Полный цикл | Дизайн, разработка, SEO, GEO, поддержка — всё под одной крышей. Без потерь на стыках между подрядчиками. |
| 5 | Код без замков | Чистый, документированный код. Если понадобится другой специалист — он подхватит проект без боли. |

### 3.4 Услуги

- Eyebrow: `Что мы делаем`
- H2: `Полный цикл разработки`
- Сетка: 5 карточек (иконка + название + краткое описание)
- Фон секции: `$color-bg`

| Услуга               | Описание                                                                |
| -------------------- | ----------------------------------------------------------------------- |
| Дизайн               | UX-исследование, прототипирование, UI-дизайн, дизайн-система            |
| Разработка           | Frontend, backend, CMS, интеграции, адаптив, скорость                   |
| SEO                  | Технический аудит, контентная стратегия, рост в поиске                  |
| GEO-оптимизация      | Оптимизация под AI-поисковики: Perplexity, ChatGPT, Google AI Overviews |
| Поддержка и развитие | Мониторинг, обновления, доработки, масштабирование                      |

### 3.5 Как мы работаем

- Eyebrow: `Процесс`
- H2: `От замысла до результата`
- Лэйаут: вертикальный степпер (как в референсе), нумерованные шаги
- Фон секции: `$color-bg-alt`

| Шаг | Название    | Описание                                                                                 |
| --- | ----------- | ---------------------------------------------------------------------------------------- |
| 01  | Погружение  | Исследуем бизнес, цели и аудиторию. Формулируем задачу точнее, чем сформулировал клиент. |
| 02  | Стратегия   | Проектируем решение, согласуем этапы, сроки и KPI.                                       |
| 03  | Дизайн      | Создаём интерфейс, итерируем с клиентом до полного согласования.                         |
| 04  | Разработка  | AI ускоряет рутину — код ревьюит человек. Результат: скорость без потери качества.       |
| 05  | Запуск      | Деплой, SEO-старт, настройка аналитики.                                                  |
| 06  | Партнёрство | Поддержка, доработки, развитие. Остаёмся рядом столько, сколько нужно.                   |

### 3.6 Портфолио

- Eyebrow: `Наши работы`
- H2: `Проекты`
- Лэйаут: сетка 3 колонки (десктоп) / 1 колонка (мобильный)
- Карточка: превью-изображение (тёмный placeholder) + название проекта + теги услуг + ссылка
- Заглушки: 3–6 карточек с единой структурой
- Фон секции: `$color-bg`
- Важно: структура карточки рассчитана на будущий реальный контент (изображение, теги, ссылка на кейс)

### 3.7 Цифры (Stats)

- Лэйаут: 4 показателя в одну строку (как в референсе)
- Фон секции: `$color-bg-alt`, тонкая разделительная линия между колонками

| Значение | Подпись                     |
| -------- | --------------------------- |
| `100%`   | проектов сданы в срок       |
| `×2`     | быстрее среднего за счёт AI |
| `≤5`     | проектов одновременно       |
| `0`      | брошенных проектов          |

### 3.8 Отзывы

- Eyebrow: `Что говорят клиенты`
- H2: `Отзывы`
- Лэйаут: сетка карточек 3 колонки (десктоп) / 1 колонка (мобильный)
- Карточка: текст отзыва + аватар + имя + компания/должность
- На старте: 3 заглушки с реалистичным текстом
- Фон секции: `$color-bg`

### 3.9 Цены

- Eyebrow: `Прозрачные цены`
- H2: `Выберите формат работы`
- Лэйаут: 3 колонки, средняя выделена акцентным бордером (как в референсе)
- Фон секции: `$color-bg-alt`

| Пакет              | Позиционирование              | Состав                                                         | Цена    |
| ------------------ | ----------------------------- | -------------------------------------------------------------- | ------- |
| Лендинг            | Для старта и проверки гипотез | Дизайн + разработка + базовое SEO                              | от X ₽  |
| Корпоративный сайт | Популярный выбор              | Полный цикл: дизайн + разработка + SEO + GEO + поддержка 3 мес | от Y ₽  |
| Индивидуально      | Для сложных задач             | Обсуждаем под конкретный проект                                | Обсудим |

> Цены — заглушки, итоговые значения вставляет клиент.

### 3.10 CTA

- H2: `Давайте создадим ваш следующий проект`
- Подзаголовок: `Расскажите о задаче — обсудим, как её решить`
- Форма в модальном окне (через существующий `Modal` + `ContactForm`):
  - Имя
  - Телефон
  - Тип проекта (select или radio: лендинг / корпоративный сайт / другое)
  - Honeypot-поле (антиспам)
- Кнопка открытия: `Обсудить проект` (primary, lg)
- Результат отправки → Telegram (через `/api/contact`)
- Фон секции: `$color-bg` с градиентным свечением (как Hero)

### 3.11 Footer

- Логотип + короткий слоган
- Навигация: те же пункты, что в Header
- Контакты: Telegram / email
- Копирайт: `© 2026 Project 42`
- Фон: `#050510` (чуть темнее основного)

## 6. Форма заявки

Адаптируем существующую `ContactForm` под новую схему:

```ts
// schema.ts
{
  ;(name, phone, projectType, _honeypot)
}
```

- `projectType`: `'landing' | 'corporate' | 'other'`
- Валидация: Zod, те же правила что в шаблоне
- Telegram-сообщение: имя / телефон / тип проекта / время (Europe/Moscow)
