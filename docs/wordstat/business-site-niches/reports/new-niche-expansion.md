## Лучшие кандидаты на добавление

1. Такси: сильный новый коммерческий хвост `заказать сайт такси`, но нужны минус-слова против вызова такси (`яндекс`, `ситимобил`, `максим`, `ко времени`, `официальный сайт`, `через сайт`).
2. Производство/завод: добавить хвосты `сайт завода заказать`, `создание сайта завода`, `создание сайта производства`, `разработка сайта для производства`. Это усиливает B2B/производственную посадочную.
3. Фирма: добавить как синонимический хвост к `сайт для компании`, но не делать отдельную посадочную без доп. регионального спроса.
4. Ремонт квартир: добавить в услуги/строительство как НЧ-хвост, отдельная посадочная возможна только после добора смежных формулировок.
5. Микро-ниши: психолог, фотограф, дизайн-студия, магазин одежды, фитнес-клуб — использовать для объявлений/FAQ/блоков, не как приоритетные страницы.
